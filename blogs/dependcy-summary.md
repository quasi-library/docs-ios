# 依赖包管理介绍

> 当我们开发“火车”的过程中难免会用到别人的“轮子”，我们制造“轮子”的过程中也想分享给别的火车制造者复用。
那么这部分代码就叫做依赖包，我们可以通过托管在三方平台供交流使用。类似于Java的Maven，Android的JCenter，Node的npm/yarn等，工作职责都是一样的


## 主流解决方案
### 1. Cocoapods
> [官方网址 https://cocoapods.org/](https://cocoapods.org/)
### 2. Swift Package Manager
> [官方源码 https://github.com/apple/swift-package-manager](https://github.com/apple/swift-package-manager)
### 3. Carthage
> [官方地址 https://github.com/Carthage/Carthage](https://github.com/Carthage/Carthage)
